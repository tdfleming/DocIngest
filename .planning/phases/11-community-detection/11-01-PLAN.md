**Main entry point:**
```python
async def build_communities(
    db: AsyncIOMotorDatabase,
    tenant_id: str,
    resolutions: list[float] | None = None,
) -> dict[str, int]:
```
Flow:
1. Use `settings.community_resolutions` if resolutions is None
2. Load all entities: `await db.entities.find({"tenant_id": tenant_id}).to_list(length=None)`
3. Load all relationships: `await db.relationships.find({"tenant_id": tenant_id}).to_list(length=None)`
4. Guard: if `len(entities) < 3` or `len(relationships) == 0`, log warning with structlog and return `{"total_communities": 0}`
5. Build igraph via `await loop.run_in_executor(None, _build_graph, entities, relationships)`
6. Run Leiden via `await loop.run_in_executor(None, _detect_communities_multi_resolution, graph, resolutions)`
7. Build `idx_to_entity = {i: ent for i, ent in enumerate(entities)}` lookup
8. For each level and community member list:
   - Skip if `len(members) < 2` (filter singletons)
   - Collect entity_ids as `[str(e["_id"]) for e in member_entities]`
   - Generate title via `_generate_community_title(member_entities, level)`
   - Collect all unique chunk_ids from member entities, cap at `settings.community_max_chunks`
   - Fetch chunk texts via `await _fetch_chunk_texts(tenant_id, chunk_ids_list)`
   - Generate summary via `await loop.run_in_executor(None, _extractive_summary, chunk_texts, settings.community_max_summary_sentences)` -- only if chunk_texts is non-empty
   - Embed summary via `await loop.run_in_executor(None, embed_texts, [summary])` -- only if summary is non-empty
   - Call `await upsert_community(db, tenant_id, {...})` with level, title, summary, entity_ids, summary_embedding
   - Track count per level
9. Post-process parent/child: For each community at level N>0, find the level N-1 community whose entity_ids overlap most. Set parent_community_id on child, add child_id to parent's child_community_ids. Do this via MongoDB updates after all communities are upserted.
10. Return stats dict: `{"level_0": N, "level_1": M, ..., "total_communities": sum}`

**Helper: _build_graph**
```python
def _build_graph(
    entities: list[dict], relationships: list[dict]
) -> tuple[ig.Graph, dict[str, int]]:
```
- Create id_to_idx mapping from `str(ent["_id"])` to vertex index
- Create undirected igraph.Graph with `n=len(entities)`
- Set vertex attributes: `g.vs["name"]`, `g.vs["entity_name"]`, `g.vs["entity_type"]`
- Deduplicate edges: if same (source, target) pair appears multiple times, sum weights. Use a dict keyed by `(min(src,tgt), max(src,tgt))` to merge.
- Add edges and set `g.es["weight"]`
- Return `(graph, id_to_idx)`

**Helper: _detect_communities_multi_resolution**
```python
def _detect_communities_multi_resolution(
    graph: ig.Graph, resolutions: list[float]
) -> dict[int, list[list[int]]]:
```
- For each resolution in resolutions (enumerate for level):
  - Call `la.find_partition(graph, la.CPMVertexPartition, resolution_parameter=resolution, weights="weight")`
  - Convert `partition.membership` to dict of community_id -> list[vertex_indices]
  - Store as `results[level] = list(communities.values())`
- Return results

**Helper: _extractive_summary**
```python
def _extractive_summary(texts: list[str], max_sentences: int = 5) -> str:
```
- Split all texts into sentences using `re.split(r'(?<=[.!?])\s+', text.strip())`
- Filter sentences shorter than 20 chars
- If no valid sentences, return joined texts[:500]
- If sentences <= max_sentences, return all joined
- Use `TfidfVectorizer(stop_words="english", max_features=5000)` to fit_transform sentences
- Score each sentence by mean TF-IDF value: `np.asarray(tfidf_matrix.mean(axis=1)).flatten()`
- Select top-k indices sorted by original order
- Return joined sentences

**Helper: _generate_community_title**
```python
def _generate_community_title(member_entities: list[dict], level: int) -> str:
```
- Sort by `mention_count` descending
- Take top 3 entity names
- Return `f"L{level}: {', '.join(top_names)}"` -- deterministic for upsert dedup

**Helper: _fetch_chunk_texts**
```python
async def _fetch_chunk_texts(
    tenant_id: str, chunk_ids: list[str], batch_size: int = 100
) -> list[str]:
```
- Get client via `await get_qdrant()`
- Collection name: `f"tenant_{tenant_id}"`
- Import `Filter, HasIdCondition` from `qdrant_client.models`
- Batch through chunk_ids in groups of batch_size
- For each batch: `await client.scroll(collection_name=..., scroll_filter=Filter(must=[HasIdCondition(has_id=batch)]), limit=batch_size, with_payload=["chunk_text"], with_vectors=False)`
- Extract `p.payload.get("chunk_text", "")` from results, filter empty
- Return texts list
